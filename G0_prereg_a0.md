# G0 — Pre-registration: deriving the MOND scale a₀ from TFT

*Written before the derivation, to fix the target and the falsifiers so we cannot rationalize a wrong answer into the right number. Discipline: no fitting; label everything; route each claim to a runnable check.*

## Target
Derive **a₀ = cH₀/2π** (≈ 1.08×10⁻¹⁰ m/s²) from TFT first principles — **parameter-free**, with the **2π traced to the S¹ geometry**, not inserted to match data.

Context: MOND *fits* a₀ as a free parameter. TFT claims a₀ is *predicted* from c (a derived scale), H₀ (measured independently), and the S¹ period. Success = showing that prediction follows from the framework, not just that the number is close.

## Success gates (pre-committed)
- **G1** — H₀ appears as a phase-cycling rate of the field, and a₀ = cH₀/2π follows structurally with the 2π from S¹. (Structural/dimensional cut allowed here.)
- **G2** — the physical input(s) G1 assumed are justified from the TFT action/cosmology, not posited.
- **G3** — the crossover acceleration equals cH₀/2π, coefficient and all.
- **G4** — the modified dynamics (interpolation law g_obs(g_N)) is derived, matching the empirical RAR shape within its scatter.
- **G5** — baryons-only fit to the Milky Way + SPARC galaxies with TFT's own a₀ and interpolation, zero free parameters.

## Falsifiers (pre-committed — any of these downgrades the claim)
1. If the crossover comes out **cH₀ with a coefficient ≠ 1/2π** → the 2π/S¹ claim is FALSE. Report the actual coefficient; do not backfit.
2. If **any step requires a fitted constant** → "derived" fails; downgrade to phenomenology (same status as MOND).
3. If it requires the **absolute values of E₀, ℓ₀** or the **G-coefficient / full quantum-gravity sector** → we hit the known wall; say so and stop.
4. If G4's derived interpolation **fits worse than dark matter** on real curves → honest negative.

## Method
Simplest first (Vic's rule): attempt G1 structurally (two-rate / Compton-scale argument) before building any cosmological field theory. Escalate only if the structural cut leaves a gap.

## Honesty rule
The number a₀ ≈ cH₀/2π is a *known coincidence* in the MOND literature (a₀ ≈ cH₀/6). Reproducing the number is NOT the deliverable. The deliverable is a **structural reason** (the 2π from S¹, the mechanism of the crossover). If we can only get the number by choosing inputs to match it, that is fitting, not derivation, and we label it so.
